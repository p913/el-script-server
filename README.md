# Сервер скриптов

Сервер предназначен для отгрузки и обновления бинарных алгоритмов (скриптов) на терминалах GalileoSky(tm). 

Процесс загрузки алгоритма терминалом детально отображается в Web-интерфейсе сервера, что особенно актуально 
при нахождении терминала в зоне низкого качества GPRS:

* процент выполнения
* время начала загрузки 
* прогнозируемое время окончания
* число запросов
* число повторных запросов
* перечень завершенных (успешно или нет) загрузок

Упомянутые торговые марки являются собственностью их владельцев.

### Как получить файл для загрузки

Бинарный алгоритм имеет расширение *.bin* и появляется в каталоге *abin* конфигуратора при сохранении. Файл с раширением
*.bin*, получаемый по кнопке "Сохранить бинарник", не подходит для отгрузки терминалу! Команда терминалу для загрузки алгоритма 
должна включать расширение файла:
```
script myscript.bin
```

### Как запустить сервер

Требования:

* java 11 или выше 
* maven

В *src\main\resources\application.yaml* укажите необходимый номер TCP-порта и путь к папке *abin* конфигуратора.

Порт должен быть доступен на внешнем "белом" ip-адресе. Порт и адрес сервера необходимо указать в настройках терминала
как *"сервер загрузки скриптов"*.

Для сборки и запуска:
```
mvn spring-boot:run
```

Статус процессов загрузки отображается в браузере по адресу: [127.0.0.1:8080](http://127.0.0.1:8080/).
